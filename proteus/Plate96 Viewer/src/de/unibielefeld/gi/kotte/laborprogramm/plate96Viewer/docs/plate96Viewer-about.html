About Plate96 Viewer

The Plate96 Viewer module provides an interactive view on 96 well microtiter plates in Proteus projects. It is able to interact with the Gel Viewer on the one hand and the Plate384 Viewer on the other to provide picking functionality between gels and microtiter plates and between microtiter plates and MALDI target plates. The 96 well plate is visualized as a grid of 96 buttons which allow the user to manipulate Well96 objects and their relations.
